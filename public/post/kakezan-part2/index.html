計算スピードを比べる

前回作った関数と今回作った関数、どちらの方が計算スピードが速いのでしょうか？

system.time()を使って計算スピードを比べてみましょう！

# 前回作った関数を用意
kakezan2 <- function(n) {
  result2 <- matrix(nrow = n, ncol = n)
  for (i in 1:n) {
    result2[i,] <- seq(i, by = i, length.out = n)
  }
  colnames(result2) <- c(1:n)
  rownames(result2) <- c(1:n)
  print(result2)
}

# 20の段まで計算させる。
